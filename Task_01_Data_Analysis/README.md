# Task 01: Data Analysis & Visualization
This project is part of the Edulumos Data Science Internship Program.

## Objective
To clean, analyze, and visualize a simple dataset using Excel/Google Sheets.

## Tools Used
- Excel

## Outcome
Performed basic data cleaning, descriptive statistics, and visualizations to extract insights.

## Dataset
The dataset used in this task is an employee dataset containing details such as age, department, position, country, salary, and experience.  
The dataset was cleaned to handle missing values, duplicates, and inconsistencies before analysis.

## Files Included
- employee_data.csv — Cleaned dataset used for analysis  
- Task1.xlsx — Excel file containing analysis and visualizations
