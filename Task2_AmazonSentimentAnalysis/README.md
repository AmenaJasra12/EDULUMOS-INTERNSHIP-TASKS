# Sentiment Analysis on Amazon Product Reviews

## Project Overview
This project performs sentiment analysis on Amazon product reviews to
classify customer feedback into Positive, Neutral, and Negative categories.
The analysis demonstrates how Natural Language Processing (NLP) and
Machine Learning can be used to extract actionable insights from large-scale
text data.

## Dataset
Amazon Product Reviews Dataset  
Dataset source link
https://www.kaggle.com/datasets/arhamrumi/amazon-product-reviews

## Technologies Used
- Python
- Pandas, NumPy
- NLTK
- Scikit-learn
- Matplotlib, Seaborn

## Key Features
- Text preprocessing (cleaning, lemmatization, stopword removal)
- Exploratory Data Analysis (EDA)
- Sentiment labeling based on review scores
- TF-IDF feature engineering
- Machine Learning model for sentiment classification
- Negative review analysis for business insights

## Results
The model successfully classified customer reviews into sentiment categories
and identified common issues mentioned in negative feedback.
